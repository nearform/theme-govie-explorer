Direction A

Card Grid

Each token is a self-contained card with large side-by-side swatches, name, values, and copy button. Spacious 3-column grid grouped by semantic intent. Best for: visual scanning, onboarding (Niamh), and color comparison.

Direction B

Full-Width Rows

Each token is a horizontal row with aligned columns: light swatch, dark swatch, name, light value, dark value, copy action. Alternating row backgrounds. Best for: rapid scanning, power users (Liam), and comparing values across tokens.

Dashboard with Sidebar

WCAG Contrast Checker

The dedicated contrast checking view. Select a background token, see all text tokens rated against it with AA/AAA badges. This is a specialized route (/contrast), not the main browsing experience. Best for: Ravi's accessibility verification journey.
